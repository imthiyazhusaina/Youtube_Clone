# 🎬 YouTube Clone

A static front-end clone of YouTube built using HTML and CSS. This project was one of my first web development projects, created during my 12th grade as I began exploring front-end development and responsive web design.

Although it does not include backend functionality or video streaming capabilities, it helped me understand the fundamentals of building modern web interfaces and strengthened my HTML and CSS skills.

To see the output, use : https://imthiyazhusaina.github.io/Youtube_Clone/youtube.html

---

## 📌 Project Overview

This project replicates the look and feel of YouTube's homepage with a focus on layout, styling, and user interface design. It was developed purely for learning purposes and to gain hands-on experience with front-end technologies.

---

## ✨ Features

* 🎨 Responsive YouTube-inspired homepage
* 📺 Video thumbnail grid layout
* 🔍 Search bar UI
* 📂 Sidebar navigation
* 📌 Top navigation/header
* 👤 Channel information cards
* 🖱️ Hover effects and modern styling
* 📱 Basic responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
